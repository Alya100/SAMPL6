TechnicalDetails:
All simulations were performed on clusters of 4 NVIDIA K80 GPUs, which together
ran 8 trajectories in parallel at a time.  All simulations used groups of 48 walkers,
that were propagated for either 10k or 100k steps between synchronization points.
#
Method:
All of the simulations were conducted starting from the structures provided. 
In each case, the results are derived from a single WExplore simulation.

Unbinding simulations: 
The initial structure was used as the initial position for all 48 walkers in 
the simulation.  Each walker was given an initial weight of 1/48.  
The trajectories were run for cycles of 10000 steps, followed by resampling 
using the WExplore2 resampling algorithm with the OpenMMUnbindingDistance 
function. The unbinding rate is calculated as the sum of weights of the 
trajectories entering the unbound state divided by the elapsed time, where the
unbound state is defined as structures where the smallest protein-ligand distance 
exceeds 10 Angstroms.  Exited trajectories were reinitialized in the bound state.

Rebinding simulations:
The structure of the first 48 exit points was recorded, and used to initialize
walkers in the rebinding simulation.  Each was given a weight proportional to 
the weight of the exit point in the unbinding simulation (normalized such that
rebinding weights sum to 1). The trajectories were run for cycles of 100000 steps,
with resampling using the WExplore2 resampling algorithm with the OpenMMRebindingDistance
function. The rebinding rate is calculated as the sum of the
weights of the trajectories that enter the bound state, which is defined as 
structures that have a ligand RMSD of less than 1 Angstrom after alignment to the 
binding site.

WExplore2 algorithm:
This has not yet been published, and might be published using a different name in the
future. It uses an all-to-all distance calculation to calculate a spread function, and
proposes merging / cloning events that will optimize the spread function. The spread
function is recalculated after the merging/cloning event, and if it increases then that
move is accepted. Merging/cloning is stopped when the spread function is maximized.
In contrast with the original WExplore function, WExplore2 does not use sampling regions 
for merging and cloning.

Distance functions:
The Unbinding Distance metric is the RMSD of the ligand after alignment of the binding 
site atoms.

The Rebinding Distance between structures A and B is calculated using their respective
RMSDs to the bound state:  d_AB = (1/RMSD_A - 1/RMSD_B). In other words, it uses knowledge
of the target bound state.

Rate estimates were calculated separately for each timepoint, by summing the weights
of walkers that had exited by that timepoint.  The volume of the simulation box was
used to calculate the on rate.  The free energy of binding was calculated as
F = 0.597*ln(koff/kon).
